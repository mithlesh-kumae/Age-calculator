# 🎂 Age Calculator

A simple and responsive **Age Calculator** web application that calculates a user's exact age in **years, months, and days** based on the date of birth entered by the user.

---

## 📌 Project Information

- **Project Name:** Age Calculator
- **Developer:** Mithlesh Kumar
- **Technology Used:** HTML5, CSS3, JavaScript
- **Project Type:** Frontend Web Application

---

## ✨ Features

- 📅 Select your Date of Birth using a date picker.
- 🧮 Calculates exact age in Years, Months, and Days.
- ⚡ Instant calculation without page reload.
- 📱 Fully responsive design.
- 🎨 Clean and modern user interface.
- ❌ Handles invalid or future dates.
